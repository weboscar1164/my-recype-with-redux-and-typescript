# My Recipe App

レシピを登録・検索・管理できるWebアプリです。  
React + TypeScript + Redux Toolkit + Firebase を使用して作成しました。

## Demo

https://my-recipe-with-redux-and-ts.web.app

## Features

- レシピ一覧表示
- レシピ詳細表示
- レシピ作成 / 編集 / 削除
- お気に入り登録
- 検索機能
- JSONインポート
- ページネーション
- Firebase Authenticationによるログイン管理

## Tech Stack

Frontend

- React
- TypeScript
- Redux Toolkit
- React Router
- Material UI

Backend / Infrastructure

- Firebase Authentication
- Firestore
- Firebase Hosting
